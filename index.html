What is a Website?

This... is a website!

(not final look, will make this more modern and aesthetic + readable)

this ‘folder’s’ window should be interactive (information will appear via hover/click, it’s scrollable, it has a menu, like most websites: basically, it’s the standard look of a basic website but all parts within it are moveable and interactable.

... and behind all these are three essential technologies: HTML, CSS, and JavaScript. Click on the other folders to learn more!

HTML

HTML stands for Hypertext Markup Language.

It is the structure of the website — headings, paragraphs, links, images, and more.

CSS

CSS stands for Cascading Style Sheets.

It’s used to style the website — colors, fonts, layouts, animations, etc.

JavaScript

JavaScript makes the website interactive.

It handles clicks, forms, animations, and dynamic updates.

What's a Website?

This... is a website!


A website is like a digital "place" you visit on the internet using abrowser (like Chrome, Safari, or Edge). Just likea shop , a library, or a bulletin board in the real world, a website can hold different kinds of information—such as text, pictures, videos, or tools you can use.


Each website has its own address (called a URL) that allows people to find and vsiit it online, whether it's for learning, shopping, socializing, or entertainment.

A website page (or web page) is a single document on the internet that you can view in a web browser like Chrome, Firefox, or Safari. It's made up of different parts that work together to show you information and let you interact with it.

In simple terms:
THink of a website as abook and each webpage is like a page in that book.

Each web page usually has:
◘ Text (like articles or product info)
◘ Images (like photos or graphics)
◘ Links (to other pages or sites)
◘ Buttons (to click for actions)
◘ Forms (to enter info like sign-ups)
◘ Multimedia (like videos or audio)

Websites come in many forms and serve different purposes. Some are informational or educational, others are e-commerce sites, blogs, social media platforms, or news portals. (Shopify, n.d.; Wix, n.d.) Regardless of the type, they all aim to deliver content, services, or functionality to their users. The home apge (or homapage) is usually the amin entry that orients visitors and links to other relevant pages (Encylopedia Britatnica).

Hypertext Markup Language

HTML (Hypertext Markup Language) is basically the building blocks of a webpage It's like giving instructions to your computer on what should appear—such as titles, paragraphs, images, links, and more. Imagine it as the skeleton of a house. It doesn't make things coloful or stylish but it makes sure everything is in the right place so the website has structure and content.

HTML is the standard markup language used to create documents on the World Wide Web. It allows authors to structure content such as text, images, videso , and hyperlinks in a way web browsers can interpret and display. HTML uses a system of tags and elements to define the various parts of a webpage, such as headings, paragraphs, lists, links, and multimedia

Cascading Style Sheets

Where websites find their style
CSS [Cascading Style Sheets] is what makes a website look good. It controls the colors, fonts, layouts, and overall visual appearance. If HTML is the skeleton, CSS is the skin, clothes, and makeup that make the website attractive and user-friendly. It tells the browser how each element should appear — whether it’s a big, bold heading, a pastel background, or smooth button animation.

Anayomy of a CSS Rule

Here’s an example of how you can change the color of a heading and center it on the page using CSS: 


                h1 {
                color: #0078d7;
                text-align: center;
                }

This CSS rule targets the <h1> element, sets its color to a blue shade, and aligns the text to the center of the page.

Neubrutalism

Also known as Neobrutalism or Neo Memphis, this is a design aesthetic that originated in the late 2010s that focuses on breaking the traditional conventions of design by using unconventional visual elements, such as contrasting color palettes and grotesque typefaces.


Coined by Lin Duncan, Neubrutalism's namesake originates from the architectural movement Brutalism, which utilized unpainted concrete and geometric designs, favoring functionality over conventional aesthetic standards; Brutalism was minimalist, utilitarian, monochrome in nature.


Although Brutalism and Neubrutalism are very different visually, they both share the core value of breaking the conventions of design and create visuals in their most basic shapes and forms.

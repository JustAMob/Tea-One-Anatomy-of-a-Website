What is a Website?

This... is a website!

(not final look, will make this more modern and aesthetic + readable)

this ‘folder’s’ window should be interactive (information will appear via hover/click, it’s scrollable, it has a menu, like most websites: basically, it’s the standard look of a basic website but all parts within it are moveable and interactable.

... and behind all these are three essential technologies: HTML, CSS, and JavaScript. Click on the other folders to learn more!

HTML

HTML stands for Hypertext Markup Language.

It is the structure of the website — headings, paragraphs, links, images, and more.

CSS

CSS stands for Cascading Style Sheets.

It’s used to style the website — colors, fonts, layouts, animations, etc.

JavaScript

JavaScript makes the website interactive.

It handles clicks, forms, animations, and dynamic updates.

Inspirations


Windows 1.0 85, Windows 95, Windows XP

Initially, we were going for a very Retro theme... but we saw the mock-up and it did not look too appealing. Everything was much too grey. The team then decided on a combined approach, blending a clean-ish modern style with the blocky, fun, and loud retro aesthetic (much to the designer's delight— hey, that's me!) that breaks the 2020s trend of copy-paste, ultra-clean modern designs (not that it's bad but...).


Neubrutalism

Also known as Neobrutalism or Neo Memphis, this is a design aesthetic that originated in the late 2010s that focuses on breaking the traditional conventions of design by using unconventional visual elements, such as contrasting color palettes and grotesque typefaces.


Coined by Lin Duncan, Neubrutalism's namesake originates from the architectural movement Brutalism, which utilized unpainted concrete and geometric designs, favoring functionality over conventional aesthetic standards; Brutalism was minimalist, utilitarian, monochrome in nature.


Although Brutalism and Neubrutalism are very different visually, they both share the core value of breaking the conventions of design and create visuals in their most basic shapes and forms.
